

## DIRETRIZES DE DESIGN E UI (NOVO)
- **Tema Padrão**: O aplicativo utiliza um **Tema Claro (Light Theme)** como padrão. Qualquer nova interface, modal, tela ou componente deve ser implementado seguindo este padrão (ex: `bg-white`, `bg-slate-50`, `text-slate-900`, `text-slate-600`), evite usar cores do espectro escuro (`bg-slate-900`, `text-white`) para preservar a consistência visual em todo o app.
